Welcome to my personal portfolio! This project showcases my skills, projects, experience, and allows visitors to view and download my resume.

## Demo

You can view the live portfolio here:

## Features

- **Responsive Design:** Works perfectly on desktops, tablets, and mobile devices.
- **Hero Section:** Eye-catching introduction with gradient background, animated text, and a download/view resume button.
- **About Me Section:** A summary of my skills, experience, and background.
- **Projects Section:** Cards showcasing my web development projects with links to live demos or GitHub repositories.
- **Experience Section:** Highlights my professional experience and key roles.
- **Contact Section:** Easy way to get in touch via email, LinkedIn, or other platforms using icons.
- **Static Header & Footer:** Header and footer stay in place while content scrolls.
- **Resume Download:** Users can view or download my resume directly from the portfolio.

## Technologies Used

- **HTML5** – Markup for structure  
- **CSS3** – Styling, responsive design, and gradient backgrounds  
- **JavaScript** – Interactive features and DOM manipulation  
- **Icons** – Font Awesome for contact section  
- **GitHub Pages** – Deployment of live portfolio
